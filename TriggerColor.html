Los efectos del trigger solo se aplican cuando el jugador pasa la línea verde de la imágen (la línea indica el recorrido de la reproducción de la música).

Color: Este trigger es usado para cambiar el color a una “ID de color” durante el transcurso del nivel. Esta ID de color puede ser asignada a cualquier objeto (excepto triggers), la cantidad máxima de ID’s de color son 999. La misma ID de color puede ser modificada varias veces durante el nivel por varios triggers, al igual que varios objetos pueden tener la misma ID de color.

Con la paleta de colores podrás seleccionar cualquier color, el cual podrás observar en tiempo real en el cuadro superior izquierdo. Con los botones de la derecha podrás: "Copy": copiar el color que esté seleccionado para después "Paste": aplicar el color copiado a otro trigger que utilize ID de color"Default":remplazar el color seleccionado por el color predefinido (normalmente blanco)



En el símbolo "+" arriba de la ID de Color, podrémos seleccionar un diferente canal de color a los cuales no se les puede asignar una ID de color, pero si podrémos editar esa clase como lo haríamos con una ID de Color. Cada uno representa el color de una parte determinada, como el "BG": Fondo, que normalmente es un degradado entre el color seleccionado y negro. "G1": Parte superior de el suelo del nivel. "G2": Parte inferior de el suelo del nivel. "Line": Línea que delimita la parte superior de el suelo del nivel. "Obj": Color predefinido al colocar objetos. "3DL": Color predefinido de los objetos que se usan para simular el 3D en las estructuras.




Con el botón resaltado se puede editar el color sin que se tenga que abrir la interfaz del Trigger Color.

Al ingresar a "Edit Object" en algún objeto que se le pueda cambiar el color, se mostrará una interfaz así, los botones "P-Col 1 y 2, sirven para que el color se asigne a los colores del ícono del jugador que esté jugando el nivel, con el "Light BG" se asiganrá como un color más claro al del color del fondo en ese momento. "Next Free" te colocará el siguiente color disponible que no ah sido editado. "HSV" Te permitirá editar más propiedades del color, pero solo se le aplicarán al o los objetos que estés editando, no se aplica a todos los objetos que tengan esa ID de Color.

La cantidad de ID's de Color disponibles en total es de 999.
